Plugin Media

Pour CakePHP
Présentation Installation Utilisation

La première chose à faire est de copier le plugin Media dans le dossier Plugin de votre application. Pour cela vous pouvez télécharger le .zip ou cloner directement le dépôt

git clone git@github.com:Grafikart/CakePHP-Media.git Media

Ensuite il vous faut charger le plugin, pour cela dans votre fichier bootstrap.php vous pouvez décommenter la ligne

CakePlugin::loadAll();

// Ou utiliser
CakePlugin::load('Media');

Voila ! le plugin est bien chargé vous pouvez maintenant importer la table qui va permettre de gérer les différents médias. Pour cela il faut utiliser la console de CakePHP et utiliser la commande

cake Media.create

Et c'est tout vous êtes prêt à utiliser le plugin !
